# Android Management API

> Reference implementation of Google's Android Management API using ExpressJS.

This project is a learning-focused implementation of the Google Android Management API. It aims to explore the platform's capabilities, document best practices, and provide a clean, modular architecture for enterprise Android device management.


---

## 🚀 Features

- Google Cloud Authentication
- Enterprise Management
- Policy Management
- Enrollment Tokens
- Device Provisioning
- Device Management
- Device Commands
- Managed Google Play
- Kiosk Mode
- Compliance Policies
- Webhooks
- Swagger Documentation
- Docker Support

---

## 🛠️ Tech Stack

- ExpressJS

- Google Android Management API
- Google Cloud
